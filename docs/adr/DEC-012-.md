# DEC-012: 安全機能

Status: Accepted  
Date: 2026-06-08

## Context

安全機能そのものをYAMLで自由に無効化できる設計は危険である。

## Decision

Fixed Safety Kernel + Profile-configurable Safety Policyを採用。

## Consequences

E-stop/watchdog/TTL/release基本インターロック等は固定し、閾値や許可条件だけをProfileで調整する。

## Notes

このADRは初期設計判断の記録であり、実装フェーズで詳細要件やインターフェースが更新された場合は追記する。
