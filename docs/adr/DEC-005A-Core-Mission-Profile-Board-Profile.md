# DEC-005A: Core / Mission Profile / Board Profile分離

Status: Accepted  
Date: 2026-06-08

## Context

Lv1〜3が主成果物であり、種コンはMission Profileの一つであるため、混在する概念を層分離する必要がある。

## Decision

Core Platform、Mission Profile、Board Profileを分離する。

## Consequences

Core Platform、Mission Profile、Board ProfileをYAML/設計上の独立概念として扱う。

## Notes

このADRは初期設計判断の記録であり、実装フェーズで詳細要件やインターフェースが更新された場合は追記する。
