# DEC-009: 認証・認可

Status: Accepted  
Date: 2026-06-08

## Context

Lv3で遠隔操作・テレメトリ・複数ローバー管理を行うため、認証・認可が必要。

## Decision

username/password + server-side session。admin/operator/viewer/phone_nodeを採用。

## Consequences

人間ユーザーとスマホOBCノードを分離し、Mission Profileによるコマンド許可/禁止を可能にする。

このADRは初期設計判断の記録であり、実装フェーズで詳細要件やインターフェースが更新された場合は追記する。
