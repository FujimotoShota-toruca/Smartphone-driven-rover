# DEC-017: リポジトリ構成

Status: Accepted  
Date: 2026-06-08

## Context

Web/Cloud/Firmware/Adapterが同一schemaを共有するため、初期polyrepoは同期コストが大きい。

## Decision

monorepo + 厳格なpackage境界。Mission Database / Protocol定義を単一正本として管理。

## Consequences

monorepo内で境界を明確にし、将来分離可能な構造にする。

## Notes

このADRは初期設計判断の記録であり、実装フェーズで詳細要件やインターフェースが更新された場合は追記する。
