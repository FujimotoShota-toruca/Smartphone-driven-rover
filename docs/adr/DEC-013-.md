# DEC-013: 手動操作と自律制御の優先順位

Status: Accepted  
Date: 2026-06-08

## Context

開発・教材では手動割込みが有用だが、Competition Profileでは外部操縦を禁止できる必要がある。

## Decision

Manual OverrideはCore Platformに持ち、使用可否はMission Profileで制御する。

## Consequences

Manual OverrideはCoreに持ち、Mission Profileで有効/無効を切り替える。

このADRは初期設計判断の記録であり、実装フェーズで詳細要件やインターフェースが更新された場合は追記する。
