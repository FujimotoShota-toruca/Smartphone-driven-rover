# DEC-015: ログ・リプレイ設計

Status: Accepted  
Date: 2026-06-08

## Context

制御履歴説明、Lv3リプレイ、将来ROS連携には構造化された時系列ログが必要。

## Decision

JSONLイベントログ + CSV export。Web地上局リプレイ、ROS/Simulator Adapterは将来拡張。

## Consequences

JSONLを正本ログとし、CSV exportとWeb replayを提供する。

## Notes

このADRは初期設計判断の記録であり、実装フェーズで詳細要件やインターフェースが更新された場合は追記する。
