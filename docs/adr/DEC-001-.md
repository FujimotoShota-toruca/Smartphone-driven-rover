# DEC-001: 対応プラットフォーム

Status: Accepted  
Date: 2026-06-08

## Context

Web Bluetoothを前提にするため、正式対応ブラウザとスマホOSを限定する必要がある。

## Decision

初期正式対応は Android Chrome。iPhone/Safari は将来拡張。

## Consequences

初期MVPはAndroid Chromeに集中し、iPhone/Safari対応はネイティブラッパー等の将来拡張に逃がす。

このADRは初期設計判断の記録であり、実装フェーズで詳細要件やインターフェースが更新された場合は追記する。
